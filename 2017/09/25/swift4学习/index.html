swift4学习

2017-09-25

本来打算学习 3 的，结果 4 马上就出来了，耽搁了一段时间，从头搞起。完了准备写一些 macOS 的应用玩，或者 iOS 的。

主流工程编程语言主要发展了四代，C 语言（命令式） -> C++/Java/C#( 静态面向对象 ) -> js/python/ruby( 动态函数式 + 面向对象 ) -> swift/kotlin/rust( 静态函数式 + 面向对象 )，最后的 swift/kotlin， 这些又被成为现代编程语言。严格来说 golang 也算是这类的，但是 golang 的设计实在太工业化，缺乏很多高级特性，更像是一门 DSL.

安装

用 macOS，装上 xcode，完事。swift 自带了 repl 和 playground 功能，后者比 ipython book 更牛逼，适合用来写文档，同时其中的代码可以直接运行。

基本语法

变量与语句

  1. 使用let声明常量，使用var声明类型推导的变量；
  2. 使用:在变量名后面声明类型。类似 haskell，所有类型名首字母均大写；
  3. String 是不可变的值类型，String 的基本构成是Character
  4. 类型转换类似 Python，format 语句倒是类似 es6 的${}，不过是\(v)这种格式；
  5. dict 和 list 都是使用[]来创建，也就是说类似 lua 的设计。声明方式：var t1=[String](), let t2 = [String: Int]()
  6. 使用Set<T>创建 set
  7. for, if的语法很像 golang，注意和 Python 不同，没有隐式的 bool；
  8. 可以使用guard来做提前退出（这个关键字同样来自 haskell）
  9. 所有变量默认为非 nil 的，如果可能为 nil，在变量名后面加?，类似其他语言中的optional或者 haskell 中的Empty
  10. 如果确定不可能为 nil，可以使用!后缀声明变量
  11. 判断是不是 nil，可以在 if 的条件中直接将带?的变量赋给一般变量，如果为nil则if语句为 false；或者用??赋值，相当于在 js 中使用||
  12. 可以使用repeat语法；
  13. 可以使用1..<3或者1...3语法，表示数字范围；
  14. switch..case可以用来实现模式匹配；
  15. 使用typealias给类型别名
  16. 使用断言 (assert) 做测试，使用前置条件 (precondtion) 做约束判断，换而言之 assert 在 release 时是不会生效的

函数

  1. 函数语法也类似 golang，但是引入了label系统，所以格式看起来很奇怪，label 用来实现 Python 中的指定名称传参
  2. 默认情况下所有函数参数是不可变的，需要改变的话需要inout修饰变量类型，如test: inout String
  3. 使用()创建 tuple
  4. 可以使用{}创建匿名闭包函数，用in来分割声明和函数体；
  5. 在闭包中可以用$0, $1代指第一个、第二个参数

类

  1. 使用init初始化（构造器）
  2. 使用self指代自己 (this)
  3. 使用deinit析构
  4. 在声明变量时可以定义get, set, willSet, didSet，后两者相当于触发器：如果变量改变，之前和之后做什么
  5. swift 的enum异常强大，除了普通的值以外，还能定义模式；使用indirect关键字可以实现递归 enum
  6. struct 和 class 的区别在于前者永远是深拷贝的（值类型），后者使用引用（引用类型）; 还有一点，struct 默认是不可变的，如果需要在方法中修改对象，需要加上mutating修饰符

协议和扩展

  1. 可以把协议（protocol）理解为其他语言中的interface
  2. extension用来扩展已有类的功能

异常和错误处理

  1. 实现Error这个协议的类均可以被抛出
  2. 使用do..catch这个语法
  3. 可以使用类似 go 中的defer语法

泛型

  1. 类似 C 系语言中的语法，使用<>标明泛型，使用where对类型进行约束
