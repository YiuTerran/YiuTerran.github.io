RestController

这个玩意儿相当于Controller + ResponseBody，表明该类的所有方法均是返回REST的数据，而非视图

入参校验

一般将入参直接转化为对应的POJO，称作VO（Value Object）。在VO上使用JSR 303来做校验，在对应的方法中使用@Valid配合@RequestBody将参数自动转化为对应的对象进行自动校验。
