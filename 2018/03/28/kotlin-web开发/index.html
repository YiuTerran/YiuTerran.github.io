kotlin web开发

2018-03-28

作为现代编程语言，kotlin 和 swift 的地位一致，不同的是 kotlin 跑在 JVM 上，注定只能成为第二选择，Java 永远是 JVM 上的首选。kotlin 的优势在于非纯 OO，更现代的设计理念，与 Java 无缝兼容，同时有 Spring 等重量级框架支持；另外在 Android 端，作为 Google 官方支持的语言。

语法要点

  1. 可变变量使用var，不可变使用val
  2. 类似 bash 的字符串模板功能：${}
  3. 类似 Python 的条件表达式：val t = if(a>b) 3 else 5
  4. 默认所有非 null，如果可能为 null，类型后面要加?，如var a:Int? = null
  5. 使用is（!is）进行类型检查，检查完以后（在检查分支以内）就没必要再进行类型转换而是可以直接用该类型的方法（这点其实是 IDE 实现的）
  6. when表达式
  7. range 表达式1..10 step 2 or 10 downTo 1 step 2，也可以用until表示后半开区间
  8. data class创建 POJO
  9. 只读 list：listOf；只读 map：mapOf
  10. 延迟计算：类型后面加上by lazy
  11. 扩展函数：直接对类扩展，如func String.test(){println('test')}，然后就可以"123".test()了
  12. 使用object语法创建单例
  13. 支持中缀函数，注意and, or都是位操作符
  14. 集合明确分为可变和不可变两种
  15. 使用===进行引用判断（指针）
  16. 支持操作符重载
  17. 使用限定符::获取函数、扩展函数的引用
